输入两个正整数，求其最大公约数。

```java
import java.util.Scanner;

public class Main {
    public static int gcd(int n1, int n2) {
        if (n1 == n2) return n1;
        int max, min;
        int res = 1;
        if (n1 > n2) {
            max = n1;
            min = n2;
        }else {
            max = n2;
            min = n1;
        }
        for (int i = min; i>=1; i--) {
            if (max%i == 0 && min%i == 0) {
                res = i;
                break;
            }
        }
        return res;
    }
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        int n1 = scan.nextInt();
        int n2 = scan.nextInt();
        System.out.println(gcd(n1, n2));
    }
}
```

辗转相除法
```java
import java.util.Scanner;
public class Main {
    public static int gcd(int n1, int n2) {
        if (n1%n2 == 0) return n2;
        return gcd(n2, n1%n2);
    }
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        int n1 = scan.nextInt();
        int n2 = scan.nextInt();
        int res = 1;
        if (n1 >= n2) res = gcd(n1, n2);
        else res = gcd(n2, n1);
        System.out.print(res);
    }
}
```
